Core Features

Built with flexibility in mind. ForgeOS adapts to your needs whether you're gaming, developing, creating, or working.

+

Hardware Agnostic

Runs efficiently on a wide range of hardware from older systems to cutting-edge builds. Minimal resource overhead ensures stability across configurations.

G

Gaming Optimized

Enhanced driver support and optimization for popular gaming libraries. Competitive performance for modern titles without bloat.

D

Developer Friendly

Comprehensive development tools, package management, and ecosystem support. Pre-configured for programming, scripting, and software development.

C

Customizable

Fully open-source architecture allows for deep customization. Modify the kernel, desktop environment, and system components to your preference.

S

Security Focused

Modern security practices including kernel hardening, secure boot support, and regular updates. Privacy-respecting design with no unnecessary telemetry.

R

Reliable

Designed for stability and uptime. Robust package management and system updates minimize disruptions to your workflow.

Technical Specifications

Built with modern standards and interoperability in mind.

KERNEL

Linux Kernel Base

Built on the latest stable Linux kernel with selective patches for performance and compatibility. Full access to kernel customization and compilation.

GRAPHICS

Multi-GPU Support

Compatible with NVIDIA, AMD, and Intel graphics. Support for both proprietary and open-source drivers with community driver packages available.

STORAGE

Modern Filesystem

Support for Btrfs, ext4, and other modern filesystems. Optimized I/O handling for both SSDs and traditional storage configurations.

DESKTOP

Flexible DE Options

Choose from GNOME, KDE Plasma, XFCE, or lightweight alternatives. Mix and match components based on your performance needs and preferences.

PACKAGE MANAGER

Robust Package Management

Multiple package manager options including DNF and Pacman variants. Easy software installation and system updates with dependency resolution.

COMPATIBILITY

Cross-Platform Tools

Full compatibility with standard Linux applications and tools. Containerization support for isolated application environments and reproducible workflows.
